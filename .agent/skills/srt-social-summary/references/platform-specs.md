# Platform Specifications

Detailed specifications for each social media platform to optimize content performance.

## Facebook

### Algorithm Insights
- Prioritizes content that sparks meaningful interactions (comments > reactions > shares)
- Native video performs better than links
- First 2-3 lines determine if users click "See More"
- Posts with 1-2 hashtags perform better than 3+
- Optimal posting times: 1-3 PM on weekdays

### Character Limits
- **Recommended**: 100-250 characters for maximum engagement
- **Maximum**: 63,206 characters
- **Preview**: First ~120 characters visible before "See More"

### Best Practices
- Use emojis strategically (1-3 per post)
- Ask questions to encourage comments
- Tag relevant pages/people when appropriate
- Use line breaks for readability
- Include clear call-to-action

### Content Types That Perform Well
- Personal stories with lessons
- Behind-the-scenes content
- User-generated content
- Polls and questions
- Video previews with context

### Hashtag Strategy
- **Optimal**: 2-3 hashtags
- Place hashtags at the end
- Mix branded + topical hashtags
- Avoid overly generic tags
- Research Facebook-specific trending tags

### Tone Guidelines
- Conversational and friendly
- Community-oriented language
- Inclusive and relatable
- Storytelling approach
- Authentic voice

## Thread (Twitter/X)

### Algorithm Insights
- Recency is critical (half-life ~18 minutes)
- Engagement in first hour determines reach
- Threads perform better than single tweets
- Media (images/video) increases engagement 35%
- Quote tweets and replies boost visibility

### Character Limits
- **Maximum**: 280 characters per post
- **Optimal**: 71-100 characters (highest engagement)
- **Thread**: Can chain multiple posts

### Best Practices
- Front-load the hook (first 5 words)
- Use thread format for longer content
- One idea per tweet
- Strategic use of line breaks
- Pin important threads

### Content Types That Perform Well
- Hot takes and opinions
- Quick tips and insights
- Thread breakdowns (numbered)
- Questions and polls
- Timely commentary

### Hashtag Strategy
- **Optimal**: 1-2 hashtags
- Use trending hashtags when relevant
- Create branded hashtags for campaigns
- Place naturally in sentence or at end
- Research real-time trending topics

### Tone Guidelines
- Concise and punchy
- Confident and direct
- Thought-provoking
- Conversational but sharp
- Personality-driven

### Thread Structure
```
1/ [Hook tweet - standalone value]

2/ [Context or problem]

3/ [Solution or insight #1]

4/ [Solution or insight #2]

5/ [Solution or insight #3]

6/ [Conclusion + CTA]
```

## YouTube

### Algorithm Insights
- Click-through rate (CTR) on thumbnail + title is critical
- Watch time and average view duration are key metrics
- First 15 seconds determine retention
- Description helps with search ranking
- Hashtags in description boost discoverability

### Character Limits
- **Title**: 100 characters (70 visible in search)
- **Description**: 5,000 characters
- **Preview**: First 100-150 characters visible before "Show More"
- **Hashtags**: First 3 hashtags appear above title

### Best Practices
- Front-load keywords in first 2 lines
- Include timestamps for longer videos
- Add links to related content
- Use chapters/sections
- Include social media links
- Add relevant playlists

### Description Structure
```
[Hook - First 2 lines, keyword-rich]

[Detailed overview - What viewers will learn/see]

[Timestamps - If video is >5 minutes]
0:00 - Introduction
1:23 - Topic 1
3:45 - Topic 2
7:12 - Conclusion

[Additional Resources]
📚 Related video: [link]
🔗 Website: [link]
📱 Follow on Instagram: [link]

[Call-to-Action]
👍 Like this video if you found it helpful
💬 Comment below with your thoughts
🔔 Subscribe for more content

[Hashtags]
#Hashtag1 #Hashtag2 #Hashtag3
```

### Content Types That Perform Well
- Tutorials and how-tos
- Listicles (Top 10, 5 Ways, etc.)
- Behind-the-scenes
- Product reviews
- Educational content
- Entertainment/storytelling

### Hashtag Strategy
- **Optimal**: 3-5 hashtags
- First 3 hashtags appear above title
- Mix broad and specific tags
- Include branded hashtags
- Research YouTube-specific tags
- Use YouTube autocomplete for ideas

### SEO Optimization
- Include target keywords in first 25 words
- Use natural language (conversational)
- Answer common search queries
- Include related terms and synonyms
- Match search intent

### Tone Guidelines
- Informative and authoritative
- Enthusiastic about topic
- Clear and organized
- Value-focused
- Professional yet personable

## Content Adaptation Strategy

When adapting the same content for multiple platforms:

1. **Core Message**: Keep consistent
2. **Format**: Adjust for platform norms
3. **Length**: Optimize for each platform
4. **Tone**: Subtle shifts in voice
5. **CTA**: Platform-appropriate actions
6. **Hashtags**: Platform-specific research
7. **Timing**: Post at optimal times per platform

### Example: Same Content, Three Platforms

**Core Message**: "5 productivity tips for remote workers"

**Facebook**:
```
Working from home but feeling unproductive? 😓

Here are 5 game-changing tips that helped me 3x my output:

✅ Time-block your calendar
✅ Create a dedicated workspace
✅ Use the Pomodoro technique
✅ Set boundaries with family
✅ Take real breaks

Which one will you try first? 👇

#RemoteWork #ProductivityTips #WorkFromHome
```

```
5 productivity hacks that 3x'd my remote work output:
```

## Mobile Optimization

### Facebook
- 90% of users on mobile
- Shorter paragraphs (2-3 lines max)
- Thumb-stopping visuals
- Easy-to-tap CTAs

### Thread
- 80% mobile usage
- Single-column format
- Quick scrolling behavior
- Tap-friendly interactions

### YouTube
- 70% of watch time on mobile
- Vertical video growing
- Mobile-friendly descriptions
- Easy navigation to related content

## Accessibility Considerations

### All Platforms
- Use clear, simple language
- Avoid excessive jargon
- Include alt text for images (when applicable)
- Use emojis meaningfully, not excessively
- Ensure good contrast in text
- Use camel case for hashtags (#RemoteWork not #remotework)
